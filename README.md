### 📋 선택필터
- 건물 선택
  - 건물의 층 선택
    - 층에 속해 있는 시설 선택
- 일정확인 버튼 누르면 선택한 시설들 전부 보여주기
- 조건 : 
  - OO관의 전체, OO층의 전체 선택한 후 다른 시설이나 층 누르면 alert창 띄운 후 전체만 선택되어 있도록하기
  - OO관의 전체 선택 후 다른 시설 필터 뜨지 않도록하기
  - 시설까지 선택하면 체크 모양 나오고 다른 층이나 건물 다녀와도 그 체크 모양 남아있도록하기
